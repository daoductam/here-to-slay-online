# 1. Context

**Problem:**
The left and right sides of the home screen and lobby are currently empty black spaces. Players, especially new ones, lack in-app instructions or card references to understand how the game works, which card classes exist, and what each card's detailed skills are in Vietnamese.

**Affected Modules:**
- `client/src/App.tsx` — Wrap the homepage form and add the game guide sidebar.
- `client/src/pages/Lobby.tsx` — Wrap the lobby layout and add the game guide sidebar.

**Non-Goals:**
- We are not modifying the in-game screen (`client/src/pages/Game.tsx`) directly for this task.
- We are not altering any backend game rules or card logic.

---

# 2. Constraints

**Language:** TypeScript (React)
**Architecture:** Component-based UI with CSS styling.

**Rules:**
- Do NOT modify unrelated components or backend code.
- Do NOT introduce any external UI libraries. Use pure React and vanilla CSS.
- Ensure the layout is responsive (side-by-side on desktop, toggleable floating drawer on mobile).

---

# 3. Conventions

**Naming:**
- Components: `PascalCase` (e.g., `GameGuide`)
- Helper data: `camelCase` (e.g., `cardDataVi`)
- Styles: Kebab-case classes (e.g., `.game-guide-container`)

**Testing:**
- Rely on visual verification and local dev building to verify zero compiler errors.

# 5. Target Architecture

**Components:**
- `GameGuide.tsx` — Container housing the "Luật Chơi" and "Thư Viện Thẻ" tabs.
- `cardDataVi.ts` — Dictionary containing all the translation data.

**Interaction Flow:**
```
App.tsx / Lobby.tsx
  → renders <GameGuide />
    → GameGuide reads translations from cardDataVi.ts
    → Renders Tabs: Rules (Static HTML) and Catalog (Interactive list/search)
```

# 8. Task Specifications

## TASK-001: Create Vietnamese Card Data Dictionary
- **Files**: `client/src/helpers/cardDataVi.ts` (create)
- **Responsibilities**: Host the translation keys for every card in the deck (Bards, Fighters, Guardians, Rangers, Thieves, Wizards, Items, Magic, Monsters, Leaders).
- **Acceptance Criteria**: Export `cardTranslationsVi` and `gameRulesVi` structures.

## TASK-002: Create GameGuide React Component
- **Files**: `client/src/components/GameGuide.tsx` (create)
- **Responsibilities**: Render the GameGuide layout with tabs. Render Rules in Tab 1, Card list with filters and search in Tab 2.
- **Acceptance Criteria**: Searching works, tabs switch successfully, selecting card shows detailed description.

## TASK-003: Create Stylesheet guide.css
- **Files**: `client/src/style/guide.css` (create)
- **Responsibilities**: Apply dark mode colors, glassmorphism panel styling, hover interactions, scrollbar style, and responsive media queries (hide panel on screens < 1024px and show floating drawer button instead).

## TASK-004: Integrate GameGuide into Homepage
- **Files**: `client/src/App.tsx` (modify)
- **Responsibilities**: Layout wrapping to support the sidebar side-by-side with the login form on desktop.

## TASK-005: Integrate GameGuide into Lobby
- **Files**: `client/src/pages/Lobby.tsx` (modify)
- **Responsibilities**: Layout wrapping to support the sidebar side-by-side with lobby status.
